# Screenshots for F-Droid

This directory should contain phone screenshots of GLTron Mobile for F-Droid listing.

## Required Screenshots:

1. **screenshot1.png** - Main menu showing "GLTron Mobile" title and "Tap to Start"
2. **screenshot2.png** - Gameplay showing light cycles, trails, and arena
3. **screenshot3.png** - Game in action with multiple AI players and trails
4. **screenshot4.png** - Game over screen showing score and restart option

## Screenshot Guidelines:

- **Resolution**: 1080x1920 (portrait) or 1920x1080 (landscape) 
- **Format**: PNG
- **Content**: Show actual gameplay, UI elements, and key features
- **Quality**: High resolution, clear visibility of game elements

## How to Capture:

1. Run GLTron Mobile on Android device
2. Use `adb shell screencap -p /sdcard/screenshot.png`
3. Pull with `adb pull /sdcard/screenshot.png`
4. Or use Android Studio's device screenshot tool
5. Rename and place in this directory
